# Nuxt 4 Site

A professional Nuxt 4 site with TypeScript, Tailwind CSS, and responsive navigation.

## Features

- Nuxt 4
- TypeScript
- Tailwind CSS
- Responsive hamburger menu
- 4 pages with icons in navigation
- Professional, reduced design with light gray theme

## Setup

Install dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Build

Build for production:

```bash
npm run build
```

## Pages

- `/` - Home
- `/about` - About
- `/services` - Services
- `/contact` - Contact
